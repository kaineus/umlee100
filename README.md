# 이엄지 포트폴리오

https://umlee100.vercel.app

콘텐츠를 디자인해, 브랜드의 가치와 이야기를 가장 매력적인 화면으로 완성하는 디자이너 이엄지의 포트폴리오입니다.

온라인 쇼핑몰 상세페이지, 이벤트 페이지, SNS 컨텐츠, 광고 이미지, 배너 등 다양한 분야의 디자인 작업물을 소개합니다.

## 작업물 카테고리

- **상세 페이지** — 온라인 쇼핑몰 상품 상세페이지 디자인
- **이벤트 페이지** — 프로모션/이벤트 랜딩 페이지 디자인
- **SNS 컨텐츠** — 인스타그램, 페이스북 등 소셜미디어용 컨텐츠 디자인
- **광고 이미지** — 팝업 광고, 배너 광고 등 온라인 광고 이미지 제작
- **배너** — 웹사이트 메인/서브 배너 디자인
- **3D 작업물** — 3D 모델링 작업물

## 사용 도구

Photoshop, Figma, Illustrator, CAD, PowerPoint
